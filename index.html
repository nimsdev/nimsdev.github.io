Redesign





Redesigning: The site was not very user friendly to begin with. When users visited the url (schoolitup.com), they would see a page with an image grid containing 90 photos. There were three different thumbnail sizes: small 45*45px, medium 90*90px, and large 145*145px. There weren't any descriptions, image tags, or user information relating to the image.



Changes

The last major upgrade was the upload system. In order to keep photos organized by school, we made each registered school a category. If this wasn't done, users could type their school name, and images would have been scattered. Example: writing Ottawa U / University of Ottawa would result in different school pages. This could have caused school pages to merge photos from different schools as well. We also added required fields to give the photos some character. Users had to make a title, write a description, select their school, and add related image tags.
